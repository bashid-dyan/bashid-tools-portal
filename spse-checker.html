Cara Penggunaan

1

Extract file ZIP ke folder manapun, lalu double-click Install.bat (sekali saja).

2

Double-click Jalankan.bat. Jangan tutup jendela CMD yang muncul.

3

Buka Chrome, ketik localhost:5000 di address bar.

4

Pilih instansi LPSE (atau ketik kode instansi), masukkan kata kunci, dan klik Cari.

5

Klik Detail pada paket untuk melihat informasi lengkap: pemenang, peserta, nilai kontrak, dan jadwal.

Pertanyaan Umum

Kenapa harus lokal? Tidak bisa online?

SPSE memblokir akses dari IP server cloud. Tools ini harus dijalankan dari komputer langsung agar bisa mengakses data SPSE.

Bagaimana cara tahu kode instansi?

Kode instansi biasanya mengikuti pola: nama kota/kabupaten + "kab"/"kota". Contoh: bangkalankab, surabayakota, sampangkab. Untuk kementerian, biasanya singkatan: pu, kemenkeu, kemenkes.

Data apa saja yang bisa dilihat?

Semua data yang tersedia di SPSE: nama paket, HPS, pagu anggaran, metode pengadaan, peserta tender, pemenang, harga kontrak, dan jadwal proses.
